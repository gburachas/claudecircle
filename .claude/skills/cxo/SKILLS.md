# CxO Executive Domain Skills

This container houses executive-level agents for strategic decisions.

## Agents in This Domain

| Agent | Role | Primary Skills |
|-------|------|----------------|
| CEO | Strategic vision, major decisions | Strategy, approvals |
| CFO | Financial oversight, budgets | Financials, cost analysis |
| CTO | Technical strategy, architecture | Tech direction |
| Liaison | Inter-container communication | A2A coordination |

## Common Workflows

1. **Approval**: Receive request → Evaluate → Approve/Reject
2. **Strategy**: Define goals → Distribute to domains
3. **Review**: Receive reports → Analyze → Adjust direction
