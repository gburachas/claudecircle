# Operations Domain Skills

This container houses the DevOps and monitoring team agents.

## Agents in This Domain

| Agent | Role | Primary Skills |
|-------|------|----------------|
| DevOps | Deployment, infrastructure, CI/CD | Docker, K8s, pipelines |
| Monitor | System monitoring, alerting | Metrics, logs, alerts |
| Liaison | Inter-container communication | A2A coordination |

## Common Workflows

1. **Deployment**: Receive from dev → DevOps deploys → Monitor validates
2. **Incident**: Monitor detects → DevOps investigates → Report to CxO
3. **Infrastructure**: CxO approves → DevOps implements → Monitor tracks
